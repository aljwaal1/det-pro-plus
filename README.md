# دفتر الديون برو صورة

نسخة احترافية خفيفة لإدارة الديون والتسديدات بدون إنترنت، مع مشاركة كشف الحساب كصورة PNG.

## المزايا
- لوحة تحكم احترافية.
- عملاء ومدينون.
- دين / تسديد / ملاحظة.
- تاريخ استحقاق للدين.
- صفحة للديون المستحقة والمتأخرة.
- كشف حساب لكل عميل.
- مشاركة كشف الحساب كصورة.
- مشاركة كشف الحساب كنص.
- نسخ كشف الحساب.
- إرسال تذكير جاهز للعميل.
- حفظ محلي داخل الهاتف.
- يعمل بدون حساب أو خادم.
- لا يحتاج GPS أو إشعارات أو صلاحيات خطيرة.
